## 🖼 トーク画像のリストア方法
1. `chats_backup` フォルダを長押しし、別の場所へ移動  
2. LINEを開き「トーク画像フォルダのバックアップ」を実行  
3. 移動したフォルダを「LEINs backup」フォルダへ戻して上書き  
4. 「トーク画像のリストア」を実行  

※データ破損防止のため、事前に以下の動画をご確認ください  
https://youtu.be/94JN4NLGdOI  

※ファイル管理アプリは以下を推奨  
https://play.google.com/store/apps/details?id=me.zhanghai.android.files

## 📱 使用方法
LINEアプリの  
**ホーム → ⚙ 設定 → 右上「LEINs」ボタン**  
から設定画面を開けます。

※Root環境では LIME アプリから設定可能  
※クローンアプリでは LIME 側からのみ設定できる場合あり  

---

### 📩 未読のまま閲覧
トーク画面のトグルまたは✉️ボタンをONにすることで  
「未読のまま閲覧」が可能です。

※返信すると既読になります
